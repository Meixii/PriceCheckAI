📚 How to Use PriceCheck AI

Step 1: Start the Services

Click "Start All" to launch both the backend server and frontend application. The backend handles web scraping and API requests, while the frontend provides the user interface.

Step 2: Check Pharmacy Status

Use the "Check Status" button to verify if pharmacy websites are online and accessible. This helps ensure the price checking functionality will work properly.

Step 3: Monitor Logs

Watch the console logs for real-time information about your application's status. You can export or copy logs for troubleshooting purposes.

Step 4: Access the Application

Once services are running, click "Open App" to launch the main PriceCheck AI interface in your default browser.

Step 5: Manage Services

Use the restart and stop buttons to manage individual services or all services at once. This is useful for troubleshooting or applying updates.
